# Keystroke Logging Demonstration

## Overview
This project demonstrates how keystroke logging works for educational
and cybersecurity awareness purposes only.

It helps understand how keyboard input can be captured at a basic level
using Python.

## Purpose
- Educational demonstration
- Cybersecurity learning
- Understanding keylogging concepts

⚠️ This project is strictly for learning and ethical use only.

## Technologies Used
- Python
- pynput library

## How It Works
The program listens to keyboard events and records the keys pressed
during execution.
